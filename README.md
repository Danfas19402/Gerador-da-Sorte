Gerador da Sorte
O Gerador da Sorte é uma aplicação web simples e intuitiva feita para sorteios aleatórios entre os participantes. Basta inserir os nomes, clicar em "Sortear" e o sistema escolhe um vencedor de forma imparcial!

Características
Interface visual colorida e responsiva

Inserção de nomes ilimitados para o sorteio

Sorteio totalmente aleatório de um dos participantes inseridos

Botões de ação estilizados com degradado

Fácil reinício do sorteio com botão "recarregar"

Sem necessidade de backend ou instalação — funciona apenas com HTML, CSS e JavaScript

Layout
Abra o arquivo index.htmlem seu navegador.

Preencha o campo "nome do participante" e clique em inserir para adicionar nomes à lista.

Após inserir todos os nomes, clique em sortear para selecionar o vencedor aleatório.

Para reiniciar o sorteio, clique em recarregar ou utilize o botão limpar para zerar a lista.
